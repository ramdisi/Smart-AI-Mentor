# 🧠 OCR-Based MCQ Test System

This is a browser-based system that allows users to **upload scanned images** or **screenshots** of MCQ question papers (without answers) and automatically:

✅ Extract text using OCR  
✅ Identify and format MCQ questions  
✅ Generate a test interface  
✅ Allow user to answer each question  
✅ Detect correct/incorrect answers and provide hints

## 🚀 Features

- 📷 Upload **multiple images** or **scanned PDFs**
- 🔍 Automatic **text extraction** using `Tesseract.js`
- 🤖 Smart **MCQ pattern recognition** using **RegEx**
- 🧪 Interactive **test interface** generated from OCR content
- ✅ Real-time **answer checking** and **feedback hints**

---

## 🧰 Tech Stack

- **JavaScript (Vanilla)**
- **Tesseract.js** (OCR)
- **Gemini API** (answer checking)
- **HTML5 / CSS3 / Bootstrap**
- **Regex** for text structure detection
- (Optional) `pdf.js` for PDF to image rendering
